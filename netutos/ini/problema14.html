PROBLEMA #14

Crear un programa que reciba nombres por teclado y los imprima en un archivo de texto.



Análisis

Planteamiento de la solución:
Abrir archivo 1. Declarar arreglo de caracteres con el identificador nombre de la persona.
2. Con un ciclo For pedir los nombres hasta que llegue al limite permitido.
3. Cerramos el archivo con la funcion fclose que recibe un puntero de tipo *FILE.

Pseudocódigo
 
Proceso Archivo de texto
1. Abrir archivo.
2. Escribir si no existe se crea.
3. Cierra archivo.
5. Imprime en el archivo de texto.
FinProceso

Código en Lenguaje C
Este programa muestra el ingreso de cadenas de caracteres y el resultado lo enviará a un archivo de texto. 
					
#include <stdio.h>
#include <conio.h>

int main()
{
   FILE *p; //apunta a un archivo
   int i;
   char nombre[100]; //arreglo de caracteres
   p = fopen("prueba.txt","w");// w es de escritura en el archivo
   
   for (i=1; i != 5; i++)  //5 menor que i; va ingresar 4 elementos.
   {
      printf("Deme un nombre: ");
      gets(nombre); //recibe cadena
      fprintf(p, "%d\t%s\n",i,nombre); //funcion de escribir en archivo   
   }
   fclose(p); //cerramos el archivo
   printf("El programa termino normalmente");
   getch();
   return 0;    
}
