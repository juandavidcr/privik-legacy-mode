PROBLEMA #3

Diseñar un programa que calcule el promedio de las calificaciones de un alumno ingresado por teclado hasta que el usuario finalice la entrada de datos. El programa deberá mostrar el resultado en pantalla.



Análisis

Planteamiento de la solución:
Se pretende encontrar el promedio de los numeros a ingrear mediante una sumatoria de numeros ingresados por el usuario, dandole una condición de detenerse hasta que se dejen de ingresar datos. Comparando la variable ingresada por el teclado hasta que sea diferente de si misma o igual a si misma ya que el ciclo controla la condición de paro.

Pseudocódigo
 
1. Proceso promedio
2.	Escribir ("Define el numero de notas");
3.	Leer N;
4.	Sum <- 0;
5.	Para i<-1 Hasta N Con Paso 1 Hacer
6.		Escribir ("Nota: "), i;
7.		Leer M;		
8.		Sum <- Sum + M;
9.	FinPara
10.	Escribir ("Suma acumulada es : "), Sum;
11.	Promedio <- Sum/N;
12.	Escribir ("El promedio es: "), Promedio;
	
13. FinProceso

Código en Lenguaje C
El programa calcula el promedio mediante el uso de ciclos en este caso el uso del ciclo while realiza la sumatoria y las veces que fueron ingresados datos es el contador para poder dividir la sumatoria entre el numero de datos ingresados. 
/*Programa que realiza el promedio*/			
#include <stdio.h>
#include <string.h> 
//uso de la libreria cadenas
#pragma hdrstop
#pragma argsused
int main(int argc, char* argv[])
{  //variables que almacenan la suma de los datos ingresados por teclado
   int suma,sumatoria;     //inicializo variable en 0.
   int contador = 0;  
   char sigue;//varible de tipo caracter
   float promedio;
   //se inicia promedio en 0
   promedio = 0;
   sigue = 's';  //variable char
   printf("Este programa calcula el promedio, hasta que oprima 'n' se dejan de ingresar elementos\n");
   //es verdadero si se compara del mismo tipo de dato
while (sigue == 's')
{ 
 printf("Ingresa dato: \n");
 scanf("%d",&sumatoria);
 printf("Si desea ingresar otro numero presiona 's' si no presiona 'n'\n");
 fflush(stdin);
 //depurar memoria del teclado
 sigue = getch();
//contador servira para saber sobre cuanto se hara la division.
 contador++;
 promedio = sumatoria+promedio;
}
   promedio = promedio/contador;
   printf("El promedio es: %.2f \n",promedio);
   //imprime el resultado final en formato de (.00) 2 decimales
   system("pause");
   return 0;
}
