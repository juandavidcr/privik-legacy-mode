PROBLEMA #13

Crear un programa que calcule del 1 al 10 la multiplicación del número que el usuario indique



Análisis

Planteamiento de la solución:
Se pretende encontrar la tabla de multiplicar del 1 al diez, del numero que el usuario indique, con el uso de un for

Pseudocódigo
 
	Proceso Principal(void)

						Escribir("     Tabla de multiplicar");
						Escribir("Dame la tabla de multiplicar: ");
						Leer m;
					    
						Para i=1;i<=10;i++
					    
							total = i*m;
							Escribir ("%d x %d= %d\n",i,m,total);
					    
						FinPara
					    
						Escribir("fin de programa ");
						
					FinProceso

Código en Lenguaje C
Este programa muestra la Tabla de Multiplicar. 
					
#include<stdio.h>
#include<stdlib.h>

int main(void)
{
    int i,m,total;
    printf("     Tabla de multiplicar\n\n\n");
    printf("Dame la tabla de multiplicar: ");
    scanf("%d",&m);
    
    for(i=1;i<=10;i++)
    {
    
     total=i*m;
     printf("\n%d x %d= %d\n",i,m,total);
    
    }
    
     printf("\n\nfin de programa ");
    getch();
    return 0;
}
