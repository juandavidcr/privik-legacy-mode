PROBLEMA #4

Crear un programa que calcule si es un año bisiesto o no.



Análisis

Planteamiento de la solución:
Un año es bisiesto cada 4 años y como solo se sabe si un numero es multiplo de 4 cuando termina en doble cero, y multiplos de 100 y 400.

Código en Lenguaje C
El programa deberá decidir si es un añ bisiesto mediante los múltiplos de 4, 400 y 100. 
//Este es un programa que determina si un año es bisiesto o no.
#include<stdio.h> //Cabeceras
void main(){
	//Declaracion de variables locales
	int year;
	char respuesta[1];
	printf("bienvenido!\n");
	printf("Este programa te pedira que ingreses un anio y te dira si es bisiesto.\n");
	do {
		printf("Escribe un año:\n");
		scanf("%i", &year);
		/*Esta es la parte de la funcion que evalua una condicion dada:
		Un año es bisiesto cuando es divisible entre 4 pero no entre 100,
		a excepción de los años divisibles entre 400 (que tambien son bisiestos).*/
		if (((year % 4) == 0) && ((year % 100) != 0) || ((year % 400) == 0)) {
			printf ("Es un anio bisiesto!\n");
		}
		else {
			printf("No es anio bisiesto.\n");
		}
		printf("Quieres ingresar otro anio? (s/n):\n");
		scanf("%s", respuesta);
	}
	while (respuesta[0] == 's');

		printf("Gracias por su preferencia!\n");
  system("pause");
}//Fin de funcion main
