PROBLEMA #10

Crear un programa que calcule el m.c.m o el m.c.d. dados 2 numeros por el usuario.



Análisis

Planteamiento de la solución:
Se pretende encontrar el m.c.m. y el m.c.d. dependiendo la desicion del usuario mediante un switch de dos numeros dados por el usuario'.

Pseudocódigo
 
Proceso McmMcd

		Escribir ("Ingrese la opcion:");
		Leer opc;
	Dimension num[10];
Segun  opc Hacer
	opcion_1:
		si n = 0 | n = 1 Entonces
			
			Escribir ("Ingrese un numero entero: ");

		FinSi
		
		si no Entonces
			
			Escribir ("Este programa solo opera con dos numeros." ); 
			Escribir ("Para sobre-escribir uno de los dos numeros" ); 
		FinSi
			
			opcion_2:
				Escribir ("Ingrese el numero: ");

int mcd (a,b) 
{ 
int res, n = 1; 


while(n<=a&&n<=b)//cicla para el m.c.d 
{ 
while(a%n==0&&b%n==0) 
{ 
	res=n; 
	n++; 
} 
	n++; 
} 


return res;//regresa el m.c.d 
}

int res;
